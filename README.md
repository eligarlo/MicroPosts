# MicroPosts Project

Post update and delete new posts to a fake rest api.
Working with babel webpack.


## Usage

### Installation

Install the dependencies

```sh
$ npm install
```

### Serve
To serve in the browser  - Runs webpack-dev-server

```sh
$ npm start
```

### Build
Compile and build

```sh
$ npm run build
```

### API
Run a fake api for development

```sh
$ npm run json:server
```
